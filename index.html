World Wide Web WorldWideWeb developer Tim Berners-Lee ( CERN ) Announcement date December 25, 1990 (33 years ago) [1] stable version 0.18 Editing in Wikidata / 1994 (30 years ago) programming language Objective C operating system NEXTSTEP language english type Web browser , web authoring tool license public domain Website www.w3.org/.../WorldWideWeb.html The world wide web browser teased on Nextstep WorldWideWeb is the world's first web browser and WYSIWYG HTML editing program . This program was first introduced by Tim Berners-Lee on February 26, 1991 , and runs on the Nextstep platform. It was later renamed Nexus to avoid conflict with the name of the World Wide Web . The source code was released into the public domain in 1993, making it free software . [2] [3]
